# Project 3 — Karachi Biryani Restaurant Menu

## Overview
A static restaurant menu website for a Pakistani restaurant built with vanilla HTML, CSS, and JavaScript.

## Features
- Responsive design (Desktop, Tablet, Mobile)
- Hero section with background image
- Menu categories: Starters, Main Course, Desserts, Drinks
- Sticky navigation bar
- Hover effects on menu cards
- Active state on nav links via JavaScript

## Tech Stack
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables, Media Queries)
- JavaScript (DOM Manipulation, Event Listeners)

## What I Learned
- HTML semantic tags (header, nav, main, section, footer)
- CSS Variables with :root
- Flexbox and Grid layout
- CSS Media Queries for responsive design
- Linking separate HTML, CSS, JS files
- JavaScript DOM manipulation
- addEventListener and classList
