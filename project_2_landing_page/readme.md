# Project 2/100 - Personal Brand Landing Page

A fully responsive personal brand landing page built with HTML5, Tailwind CSS, and Vanilla JavaScript.

## 🛠️ Tech Used
- HTML5
- Tailwind CSS (CDN)
- Vanilla JavaScript

## ✨ Features
- Responsive Navbar with mobile hamburger menu
- Hero section with CTA buttons
- Services section with 3 cards
- About section with stats
- Contact section with email and WhatsApp
- Smooth scroll animations
- Navbar shadow on scroll
