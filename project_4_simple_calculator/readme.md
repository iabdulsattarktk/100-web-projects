# Project 4 — Simple Calculator

## Overview
A clean, modern web-based calculator built with vanilla HTML, CSS Grid, and JavaScript. It features an interactive UI with dynamic display updates for mathematical expressions and results.

## Features
- Fully functional basic arithmetic operations (+, -, *, /)
- Responsive layout using CSS Grid for perfect button alignment
- Double display system (shows current expression and dynamic result)
- Hover and active states for interactive button feedback
- Clear functionality (C) to reset data instantly

## Tech Stack
- HTML5
- CSS3 (CSS Grid, Flexbox, Box Shadows, Transitions)
- JavaScript (DOM Manipulation, Event Listeners, Query Selectors, eval)

## What I Learned
- Structuring complex components using HTML semantic layers
- Implementing modern UI styling with CSS Grid (`grid-template-columns`, `span`)
- Smooth UI state transitions using CSS `:hover` pseudo-classes
- Managing application state via JavaScript variables (`expression`)
- Listening to bulk elements using `querySelectorAll` and `forEach` loops
- Safe dynamic evaluation of string-based mathematical expressions using `eval()`
